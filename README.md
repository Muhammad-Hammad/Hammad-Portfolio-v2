# Hammad Ghani — Senior Frontend Engineer Portfolio

A premium, dark-themed personal portfolio built with Next.js 14, Tailwind CSS, Framer Motion, and ShadCN/UI.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Components:** ShadCN/UI (customized)
- **Icons:** Lucide React
- **Fonts:** Inter (via `next/font`)

## Getting Started

### 1. Install Dependencies

```bash
npm install
```

### 2. Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 3. Deployment

This project is ready to be deployed on Vercel.

```bash
vercel deploy
```

## Features

- **Premium Dark Mode UI:** Custom color palette with deep indigo and violet accents.
- **Framer Motion Animations:** Smooth page transitions, scroll reveals, and interactive hover states.
- **Interactive Cursor Glow:** A subtle radial gradient that follows the mouse cursor.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewports.
- **SEO Optimized:** Proper metadata and semantic HTML structure.
